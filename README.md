# Deep Learning Based Object Detection using MobileNet SSD and OpenCV

## Project Overview

This project demonstrates how to perform object detection using a pre-trained MobileNet SSD (Single Shot MultiBox Detector) model with OpenCV's Deep Neural Network (DNN) module.

Unlike traditional image classification, object detection not only identifies what objects are present in an image but also determines their exact locations by drawing bounding boxes around them.

The MobileNet SSD model used in this project is trained on the COCO (Common Objects in Context) dataset and is capable of detecting multiple everyday objects such as people, cars, bicycles, traffic lights, animals, fruits, and many more.

The primary goal of this project is to understand the complete object detection pipeline, including model loading, image preprocessing, inference, confidence filtering, and visualization of detection results.

---

## What is Object Detection?

Object Detection is a Computer Vision task that combines:

1. Object Localization

   * Finding where an object is located in an image.

2. Object Classification

   * Identifying what the detected object is.

For example, in a traffic scene, an object detector can identify:

* Cars
* People
* Traffic Lights
* Buses
* Motorcycles

and simultaneously draw bounding boxes around each detected object.

---

## Why MobileNet SSD?

MobileNet SSD is a popular object detection architecture because it provides a good balance between speed and accuracy.

### MobileNet

MobileNet acts as the backbone network.

Its responsibility is to extract useful visual features from the input image such as:

* Edges
* Corners
* Shapes
* Textures
* Object parts

MobileNet is lightweight and computationally efficient, making it suitable for real-time applications.

### SSD (Single Shot Detector)

SSD performs:

* Object Classification
* Bounding Box Prediction

in a single forward pass through the network.

This makes SSD significantly faster than traditional region-based detectors.

## Project Workflow

The complete object detection pipeline follows the steps below:

### Step 1: Load the Pre-trained Model

The TensorFlow frozen graph and configuration file are loaded into memory.

```python
net = cv2.dnn.readNetFromTensorflow(modelFile, configFile)
```

---

### Step 2: Read Input Image

The input image is loaded using OpenCV.

```python
img = cv2.imread("image.jpg")
```

---

### Step 3: Create Blob

The image is converted into a blob before being passed to the network.

```python
blob = cv2.dnn.blobFromImage(
    img,
    scalefactor=1.0,
    size=(300,300),
    mean=(0,0,0),
    swapRB=True
)
```

The blob operation:

* Resizes the image
* Normalizes pixel values
* Rearranges image channels
* Converts the image into a format accepted by the neural network

---

### Step 4: Perform Inference

The blob is passed through the network.

```python
net.setInput(blob)
detections = net.forward()
```

The network predicts:

* Object class
* Confidence score
* Bounding box coordinates

---

### Step 5: Decode Predictions

Each detection contains:

* Class ID
* Confidence Score
* Bounding Box Coordinates

Only detections above a predefined confidence threshold are considered valid.

---

### Step 6: Draw Bounding Boxes

Bounding boxes and class labels are drawn on the image.

```python
cv2.rectangle(...)
cv2.putText(...)
```

This allows easy visualization of detected objects.

---

## Understanding Confidence Scores

Every prediction generated by the network contains a confidence score between 0 and 1.

Example:

| Object | Confidence |
| ------ | ---------- |
| Apple  | 0.95       |
| Orange | 0.88       |
| Person | 0.91       |

Higher confidence indicates greater certainty in the prediction.

In this project, different confidence thresholds were tested:

### Low Threshold (0.0)

* Detects almost everything
* Produces many false detections

### Medium Threshold (0.3)

* Balanced detection quality
* Good trade-off between precision and recall

### High Threshold (0.9)

* Very strict filtering
* Keeps only highly confident detections
